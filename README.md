# Aura AI – Mental‑Health Companion

A lightweight, web‑based mental‑health companion built with vanilla HTML, CSS and JavaScript.  
It provides:

- A soothing **Aura Wave** visualizer with ambient sound controls.
- A **chat‑bot** that answers mental‑health‑related questions (no medical advice).
- A **mood‑tracker** and **breathing‑coach** to help users relax.
- Multilingual interface (English, Arabic, Hindi, Urdu, etc.).

The project is completely client‑side – no server or database is required.  
You can run it locally with `python -m http.server 8000` or host it on GitHub Pages.

---

## Getting Started
1. Open `index.html` in a browser.
2. Use the navigation drawer (top‑right ☰) to switch languages or explore sections.
3. Adjust the Aura Wave sliders to change the wind and rain sounds.
4. Interact with the chatbot at the bottom‑right corner.

---

## Deploying to GitHub Pages
1. Create a GitHub repository (e.g., `username/aura-ai`).
2. Add the three files (`index.html`, `styles.css`, `app.js`) plus this `README.md`.
3. Enable **GitHub Pages** in the repository settings – select the `main` branch and `/` (root) folder.
4. After a few minutes, GitHub will give you a live link like `https://username.github.io/aura-ai/`.
